# Rinon-Lootbug-mod
change the Lethal Company Hoarding Bug chitterSFX to Rinons famous "Hellooooo"

## Manual Install Instructions
1. Install BepInEx
2. Extract Rinhello.dll and rinonhello into the Plugins folder of BepInEx
